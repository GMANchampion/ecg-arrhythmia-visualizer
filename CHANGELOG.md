# Changelog - v1.1.0

### Added/changed

- Added rhythm explanation and clue panels.
- Added readouts for rate class, P waves, and QRS width.
- Cleaned up the rhythm buttons so the app feels more like a learning tool.
- Kept the original modulo-based waveform timing v1.0.0.

### Still rough

- AFib irregularity is still a rough visual effect rather than true irregular R-R scheduling.
- No PVC or V-Tach modes yet.
- The heart-rate display is a fixed estimate for each rhythm.
